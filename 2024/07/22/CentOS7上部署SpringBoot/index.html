4. 设置 Spring Boot 应用程序

4.1 运行 Spring Boot 应用程序

您可以使用 nohup 命令在后台运行 Spring Boot 应用程序。确保您在应用程序目录中，然后运行以下命令：

  • nohup: 允许您在关闭终端后继续运行应用程序。
  • > /path/to/logfile.log: 将输出重定向到指定的日志文件。
  • 2>&1: 将错误输出重定向到同一日志文件。
  • &: 将任务放入后台运行。

您可以使用 jobs 命令查看后台进程，并使用 fg 将其恢复到前台。
